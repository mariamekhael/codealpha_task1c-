# Student CGPA Calculator (C++)

## Features
- Input the total number of courses taken.
- Store course details including Name/ID, Grade Point, and Credit Hours.
- Automatically calculate total credit hours and total grade points.
- Compute the final CGPA using the standard weighted formula.
- Display a formatted summary table of all courses and the final result.

## Calculation Logic
The CGPA is calculated using the following formula:
\[ \text{CGPA} = \frac{\sum (\text{Grade Point} \times \text{Credit Hours})}{\sum \text{Total Credit Hours}} \]
